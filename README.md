# Login Page

A simple and responsive login page built with **pure HTML & CSS** (no framework).

## Features

- Responsive layout (Flexbox)
- Clean & modern UI
- Easy to integrate with backend systems

## Usage

1. Copy the `index.html` file
2. Open it in your browser
3. Integrate with your backend (PHP / CI3 / etc.)
